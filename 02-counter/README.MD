# Counter App

A simple, persistent counter built with vanilla HTML, CSS, and JavaScript. It features adjustable increment/decrement steps, visual feedback via a dynamic meter, and data persistence using `localStorage`.

## ✨ Features
- **Main Counter**: Increment, decrement, or reset the value (range: -100 to 100).
- **Adjustable Step Size**: Change the increment/decrement gap from 1 to 10 using the bottom controls.
- **Visual Meter**: A colorful bar that changes hue from red to green based on the current value.
- **Persistence**: Your counter value and step size are automatically saved in the browser's `localStorage` and restored on reload.
- **Clean UI**: Uses Font Awesome icons and responsive design.
